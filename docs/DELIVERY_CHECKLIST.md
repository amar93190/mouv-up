# Checklist de rendu (client / DCX)

## 1) Dépôt GitHub structuré

- [ ] README complet à jour
- [ ] Documentation d'onboarding disponible
- [ ] Documentation sécurité/configuration disponible
- [ ] Documentation automatisations disponible
- [ ] Historique git propre (commits lisibles)

## 2) Automatisations JSON

- [ ] Dossier `automations/` présent
- [ ] Payload exemple fourni
- [ ] Schéma JSON fourni

## 3) Accès et configurations

- [ ] Variables d'environnement listées
- [ ] Aucune clé sensible committée
- [ ] `.env.example` à jour
- [ ] Instructions de setup Supabase documentées

## 4) Guide de prise en main

- [ ] Installation locale documentée
- [ ] Initialisation base documentée
- [ ] Vérifications fonctionnelles listées
- [ ] Build et preview documentés

## 5) Validation finale

- [ ] `npm run build` passe
- [ ] Parcours public validé
- [ ] Parcours auth validé
- [ ] Parcours admin/organizer validé
- [ ] Déclenchement webhook validé
